### FastCopy Auto Backup Script
一個 Windows 批次腳本，可自動將任意資料夾備份到指定位置，每次備份資料夾以日期時間命名，方便追蹤歷史版本，支援增量備份及 UTF-8 中文檔名處理，可備份到本機硬碟、網路共用或外接儲存裝置。

## 需求
- Windows
- FastCopy.exe

## 使用方法
1. 修改來源資料夾路徑
2. 修改備份目的地路徑
3. 執行 .bat 腳本

## 範例
來源: C:\Users\user\databackup
目的地: C:\Users\user\test2025122710

## 注意事項
- 建議先在非重要資料夾測試
- 需安裝 FastCopy
